    数组与切片

    数组

    包含同类型元素的集合。

    创建数组

    1. 通过：=对数组进行声明和初始化：arr := [3]int{1,2,3}

    2. 省略数组长度：arr := [...]{1,2,3,4,5}，Go会在编译器自动计算数组长度

    3. 初始化时没有填满所有数据，空位会由该类型的空值表示

      go arr := [5]int{1, 2, 3} fmt.Println(arr) // [1 2 3 0 0]

    4. 指定下标位置的元素值

    arr := [5]int{1:100, 3:300}
fmt.Println(arr)
// [0 100 0 300 0]
    1. 数组长度在定义后不课更改，可以通过len()获取数组长度

    访问数组元素

    1. 通过下标访问

    2. 遍历数组

    for i := 0; i < len(arr); i++ {
    fmt.Println("Element", i, "of arr is", arr[i])
}
    1. 通过range快速遍历
    for i,v := range arr {
    fmt.Println("Element", i, "of arr is", v)
}

// 不获取索引
for _, v:= range arr {
    // ..
}

// 只获取索引
for i := range arr {
    // ...
}

    数组是值类型

    数组变量被赋值时，会获取原数组的==拷贝==，新数组中的元素改变不会影响原数组

    a := [...]string{"USA", "China", "Germany", "France"}
b := a
b[0] = "Japan"
fmt.Println("a is :", a)
// a is : [USA China Germany France]
fmt.Println("b is :", b)
// b is : [Japan China Germany France]

    将数组作为参数传递给函数

    func main() {
    num := [...]int{1, 2, 3, 4, 5}
    fmt.Println("before: ", num)
    change(num)
    fmt.Println("after: ", num)
}

func change(num [5]int) {
    num[0] = 1000
    fmt.Println("inside function", num)
}

/*
before:  [1 2 3 4 5]
inside function [1000 2 3 4 5]
after:  [1 2 3 4 5]
*/

    切片

    长度动态变化的数组。由三部分组成：指针、长度和容量。指针指向数组起始下标，长度对应切片中元素的个数，容量则是从切片指针所指的元素开始到数组结尾的长度，切片长度不会大于容量。

    创建切片

    1. 通过array[start:end]创建
    a := [5]int{1, 2, 3, 4, 5}
slice_a := a[1:4]
fmt.Println(slice_a)
// [2 3 4]
    1. 通过make([]type,len, [capacity])创建
    // 省略了容量
Slice1 := make([]int, 3)
// [0 0 0]

// 创建一个int切片，初始元素个数为5，容量为10
Slice2 := make([]int, 5, 10)
fmt.Println(Slice2)
// [0 0 0 0 0]

    访问切片元素

    1. 遍历数组切片

    go for i := 0; i < len(slice); i++ { fmt.Println(slice[i]) }

    1. 用range快速遍历
    for i, v := range slice {
    fmt.Println("slice[ ", i, "] =", v)
}

    动态增加元素

    通过len()和cap()可以获得切片的长度和容量，通过append()可以向切片追加新元素。当切片中的len>cap的时候，切片会自动扩容为原来切片容量的两倍，如果还不够容纳全部元素，则会继续扩容。

    func main() {
    slice := make([]int, 5, 10)
    fmt.Println("len(slice)", len(slice)) // 5
    fmt.Println("cap(slice)", cap(slice)) // 10

    // 通过append添加元素
    slice = append(slice, 10, 11, 12, 13, 14)
    fmt.Println("after append len(slice)", len(slice)) // 10
    fmt.Println("after append cap(slice)", cap(slice)) // 10

    // 超出容量
    slice = append(slice, 1,2,3,4,5)
    fmt.Println("overstep len(slice)", len(slice)) // 15
    fmt.Println("overstep cap(slice)", cap(slice)) // 20
}

    内容复制

    通过copy()将一个数组切片复制到另外一个数组切片，如果切片大小不一样，则会按照小的切片复制。

    func main() {
    slice1 := []int{1, 2, 3, 4, 5}
    slice2 := []int{7, 8, 9}

    copy(slice2, slice1)
    fmt.Println(slice2) // [1 2 3]

    slice1 = []int{1, 2, 3, 4, 5}
    slice2 = []int{7, 8, 9}
    copy(slice1, slice2)
    fmt.Println(slice1) // 5 6 7 4 5
}

    共享底层

    当创建切片的时候，指针指向底层数组

    slice[1:4]的时候指针起始位置发生变化

    如果多个切片共享同一个数组的时候，每个切片的改变都会影响到其他切片

    func main() {

    array := [...]int{1, 2, 3, 4, 5}
    fmt.Println(array) // [1 2 3 4 5]

    slice1 := array[0:3]
    fmt.Println("slice1 before: ", slice1) // slice1 before:  [1 2 3]

    slice2 := array[2:5]
    fmt.Println("slice2 before: ", slice2) // slice2 before:  [3 4 5]


    slice1[2] = 1000
    fmt.Println("slice1 after: ", slice1) // slice1 after:  [1 2 1000]
    fmt.Println("slice2 after: ", slice2) // slice2 after:  [1000 4 5]

}
